## 🚀 About Me (the lore)

I'm a **vibe coder** who somehow ships **production SaaS platforms, e-commerce sites, and mobile apps** powered by AI, caffeine, and a dangerous amount of confidence. I also do **FPGA design, Verilog, and hardware acceleration** because apparently software wasn't enough chaos.

I build across the entire stack from **React/SvelteKit** frontends to **Node.js/Supabase** backends, throw it on **VPS + Vercel**, and pray to the demo gods. *It usually works.*

- 🔭 **Building:** Production apps that real humans actually use (scary, I know)
- 🤖 **Vibe Coding With:** AI that writes my code while I take the architectural credit
- 🌱 **Currently Learning:** Advanced **Verilog design & verification**, AI/ML because I hate free time
- 💬 **Ask me about:** How I shipped 6 production apps and still can't center a div without Googling
- ⚡ **Fun fact:** My code works first try. I just choose to run it multiple times for fun.
